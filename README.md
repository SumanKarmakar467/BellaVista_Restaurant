# 🍽️ BellaVista Restaurant

A responsive restaurant landing page built with pure HTML and CSS. Features a hero section, menu display, about section, and contact details with smooth layout and warm visual design.

## 🔗 Live Demo
[suman-bella-vista.vercel.app](https://suman-bella-vista.vercel.app/)

## 🛠️ Tech Stack
- HTML5
- CSS3 (Flexbox, responsive design)

## ✨ Features
- Fully responsive across mobile, tablet, and desktop
- Clean restaurant-style UI with hero banner
- Menu section with dish cards
- About and contact sections

# Just open index.html in your browser — no setup needed
